# About the MOVETHIS Folder

## What is it?

The `MOVETHIS` folder contains **all the decompiled Minecraft 1.21.10 source code** extracted from this project.

## Why does it exist?

This folder was created to make it easy to copy the decompiled Minecraft source code to a separate repository for mod development.

## What's inside?

- **net/minecraft/** - Complete Minecraft game source code (~31MB)
- **com/mojang/** - Mojang utility libraries (~76KB)
- **README.md** - Documentation about the contents

## How to use it?

Simply **drag and drop** or **copy** the entire `MOVETHIS` folder to your new project repository. 

You can:
1. Keep the folder name as `MOVETHIS`
2. Rename it to something like `minecraft-source`
3. Extract the contents (`net/` and `com/` folders) directly to your project root

## Total Size

~31MB of decompiled Java source code
